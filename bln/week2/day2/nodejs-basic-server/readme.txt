# Basic Node.js Server Project

## Setup Instructions

1. Clone the repo or unzip the folder
2. Run `npm install`
3. Copy `.env.example` to `.env` and set the port
4. Start server:
   - Development: `npm run dev`
   - Production: `npm start`

## Scripts

- `npm run dev`: Starts server with nodemon
- `npm start`: Starts server normally

## Features

- Custom HTTP server with route handling
- .env config
- Logger with timestamps
- Error handling for 404 pages
- Debugging practice
